+ 安装语言服务器 =lsp-install-server=
=install python language server= 在win32系统上，直接调用 =lsp-install-server= 会报错，
因为自动下载了64位服务器。一个方法是使用vscode下载 _python language server_ ，在
=C:\Users\\wjb\AppData\\Local\Microsoft\\Python\\ Language Server=, 然后建立一个快捷方
式到 =emacs.d/.local/etc/lsp/mspyls/=
[[file:c:/Users/wjb/appdata/local/Microsoft/Python Language Server/][Python language server in User Local AppData]]
